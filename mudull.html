RUN MUDULL RUN

We create an addictive game to let users control the lovely Mcmug to collect green seeds.

To drive organic and viral growth, the game is connected to Facebook so that users can ask their real friends for help and share "treasures". To make the game even more interactive, the hidden stage of the game can only be unlocked by an on-site location check-in, a feature that will bridge the gap between online and offline experience and drive online users to the event.
